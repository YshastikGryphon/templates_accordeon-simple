# Простой аккордеон

Самый банальный аккордеон (Раскрывающийся список).

Группа элементов аккордеона находится в js-accords_one / js-accords_mult
- .js-accords_one - закрывает другие элементы при открытии
- .js-accords_mult - без ограничений

Элемент аккордеона - .js-accord. Внутри:
- .js-accord-head - открывает закрывает элемент
- .js-accord-body - основная обёртка (здесь задается высота 0)
- .js-accord-content- высота внутреннего контента (отсюда берется высота. Не изменять!)
